## 🚀 技术栈

该博客基于以下技术构建：
- **静态网站生成器**: [Hugo](https://gohugo.io/)
- **主题**: [PaperMod](https://github.com/adityatelange/hugo-PaperMod)
- **部署与自动化**: [Netlify](https://www.netlify.com/)

## ✍️ 内容板块

博客内容主要涵盖以下几个维度：
- **深度探讨**: 对一个问题的深度研究。
- **商业模式**: 研究AI时代的新商业模式。
- **认知分享**: 对于认知升级的一些思考。

## 🛠️ 本地部署

如果你想在本地运行此博客，请参考以下步骤：

1. **克隆仓库**

```bash
git clone [https://github.com/seeyoumeet/blog.seeyoumeet.top.git](https://github.com/seeyoumeet/blog.seeyoumeet.top.git)
cd blog.seeyoumeet.top
hugo server
```

2. **启动本地服务器**
   
```bash
hugo server -D
```

3. **新建文章**
```bash
hugo new posts/my-new-post.md
```
